# ♟️ Trabajo de Informática Industrial - ARCHON

Este proyecto consiste en el desarrollo de una versión personalizada del clásico juego **Archon**, combinando estrategia en tablero y combate en tiempo real. El desarrollo se está realizando en **C++** utilizando la librería gráfica **ETSIDI**.

## 📝 Descripción del Proyecto
La aplicación recrea la lógica de duelo entre dos bandos. El juego se divide en dos fases principales:
1. **Fase de Tablero:** Movimiento estratégico de las piezas por turnos.
2. **Fase de Arena:** Combate de acción cuando dos piezas coinciden en la misma casilla.

La versión se ha adaptado para que los bandos enfrentados sean los turnos de mañana y tarde del grado de Ingeniería Electrónica Industrial y Automática de la ETSIDI. 

Ambos bandos vienen representados por los profesores de la asignatura de Informática Industrial y Comunicaciones.
